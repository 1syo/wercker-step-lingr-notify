# Lingr notification step

## Options

* ``room-id`` (required)
* ``bot-id`` (required)
* ``secret`` (required)
* ``message`` 

## Example

```
- 1syo/lingr-notify@0.0.1:
    room-id: YOUR_ROOM_ID
    bot-id: YOUR_BOT_ID
    secret: YOUR_SECRET
    message: Deploy it!
```
